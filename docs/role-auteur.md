# Rôle de Zidane Djamal A

**Zidane Djamal A** est positionné dans ce dépôt comme **Architecte Oracle Infrastructure, HA/DR et formateur technique**. Son rôle pédagogique consiste à organiser le savoir Exadata autour d’un angle infrastructure, c’est-à-dire en partant des contraintes datacenter, réseau, stockage, compute, sécurité et exploitation avant de relier ces éléments aux workloads Oracle Database.

| Responsabilité | Description |
|---|---|
| Vision pédagogique | Structurer le parcours pour des DBA, architectes et administrateurs systèmes. |
| Architecture infrastructure | Expliquer les composants Exadata, le réseau, les cells, ASM et les choix de design. |
| HA/DR | Relier Exadata aux architectures RAC, Data Guard, MAA et aux tests de reprise. |
| Labs | Fournir des scénarios de pratique réalistes, sûrs et vérifiables. |
| Runbooks | Documenter les opérations courantes, patching, diagnostic et escalade support. |

> Le rôle n’implique pas une affiliation avec Oracle. Les produits et marques Oracle restent la propriété d’Oracle et de ses ayants droit.
